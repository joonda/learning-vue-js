## 노트

### 1. Vue CLI 소개 및 설치
```terminal
npm install -g @vue/cli
```

```terminal
vue create project-name
```
* 이후 [Vue 3] 선택.

### 2. Vue 프로젝트 생성 과정 설명 및 서버 실행
```terminal
cd project-folder
```

```terminal
npm run serve
```

### 3. Vue 프로젝트 폴더 내용 살펴보기
* `ESlint`
    * 자바 문법 감시 도구
* `public` > `index.html`
    * index.html이 애플리케이션의 진입점

### 4. 라이브러리, 파일 임포트 방식 설명
* npm 기반으로 프로젝트를 진행

### 5. 페이지 로딩 과정 분석
